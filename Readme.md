# 项目介绍
一个openttd脚本，让游戏更具有挑战性！ 由`递归岛`开发

# 目标
## 新的税法
根据每家公司的市值，为每家公司制定独有的税率，具体来说，计算公司市值相与全部公司平均市值的比值，乘以基准税率，就是该公司的本季度税率。

## 环保主义
由于飞机对环境的破坏太大，政府将在年末对全部飞机所获得的收益单独征收特定比例的环保税，该税与上方所说的市值税无关

## 基于24h时间动态客流分配
基于`Peaks and Thoughs`脚本实现相关功能。

## 团队
脚本通过公司id分配公司团队。不同团队之间的列车不能够停靠对方的车站，列车可以在自己的团队的其他公司的车站停靠（开启基建共享）

## 城市发展
基于`Renewed Village Growth`实现城市发展控制。
